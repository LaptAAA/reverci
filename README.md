<h3 align="center">Работа с программой</h3>

Игра предоставляет возможность играть как двум реальным игрокам, так и в одиночном режиме с компьютером. Игра с компьютером включает в себя два уровня сложности.  
При игре с компьютером есть возможность откатить свой ход хоть до первоначального расположения фишек. В игре на двоих откатывать ходы нельзя.
